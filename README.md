# Flask Notes App

## Description
A simple Notes application built using Flask. This project was created by following a Flask tutorial to understand the fundamentals of backend web development and Flask application structure.

## Purpose
The primary goal of this project was learning and practice. Through this app, I focused on:
- Flask application structure
- Routing and request handling
- Jinja templating
- Basic CRUD operations
- Using SQLite with Flask

## Features
- Create notes
- View saved notes
- Delete notes
- Simple and clean interface

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS

This project is complete and serves as a learning exercise.
